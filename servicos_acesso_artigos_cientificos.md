# Serviços de Acesso a Artigos Científicos

Pesquisadores sem vínculo institucional podem recorrer a serviços de assinatura que oferecem acesso *ilimitado* (streaming) a artigos científicos. Dentre eles, destaca-se o **DeepDyve** ([https://www.deepdyve.com](https://www.deepdyve.com)), que funciona como um “Spotify de artigos acadêmicos”. Assinantes têm leitura **online ilimitada** de mais de **30 milhões** de artigos pagos (e 30M de acesso aberto) provenientes de \~15 mil periódicos. O plano *Pro* custa cerca de US\$499/ano (ou \~\$40/mês) e permite buscar e visualizar trechos completos sem limite diário – a única restrição é que não se pode baixar ou imprimir diretamente.

Outra opção são as assinaturas profissionais dos próprios editores. Por exemplo, o **IEEE Xplore** oferece o *IEEE Member Digital Library*, que concede a membros IEEE acesso completo a **todos** os periódicos, transações e anais de conferências da IEEE. Isto significa texto integral ilimitado de publicações IEEE (medidas, eletrônica, computação etc.) para quem mantém a assinatura anual do IEEE. Similarmente, a **ACM Digital Library** permite, via assinatura institucional ou associação à ACM, acesso sem restrições a todo o acervo da ACM (artigos, conferências, boletins).

Em cada caso, vale notar que **as assinaturas autoritárias podem ser caras** e geralmente se justificam apenas para uso intenso (múltiplos artigos por mês). No Brasil, pesquisadores também contam com o **Portal de Periódicos CAPES** (acesso gratuito) que abrange milhares de títulos das principais editoras (Springer, Elsevier, Wiley etc.), mas esse serviço é destinado a universidades públicas. Também vale explorar recursos como repositórios institucionais e redes acadêmicas (e.g. ResearchGate), além de solicitar artigos diretamente aos autores. Mas para **acesso imediato e ilimitado pago** aos maiores periódicos de computação/IA, os serviços acima são os principais exemplos legais de “streaming” científico.

## Ferramentas e APIs de Busca Avançada

Para montar um hub educacional com busca e filtros avançados, há diversas ferramentas e APIs especializadas em literatura científica. Essas soluções permitem pesquisar metadados (título, autores, resumos), filtrar por ano, área ou periódico, e obter links diretos aos artigos. Abaixo comparamos algumas das principais:

* **CrossRef REST API** ([https://api.crossref.org](https://api.crossref.org)) – API gratuita de metadados que cobre bilhões de registros DOI de artigos, livros e conferências. Permite consultas via parâmetros (autor, ano, revista, DOI, etc.) e retorna DOIs e referências. É base para muitos outros serviços e fornece links resolvíveis via DOI.
* **OpenAlex API** ([https://docs.openalex.org](https://docs.openalex.org)) – Catálogo aberto de mais de 242 milhões de trabalhos acadêmicos (inclui artigos, conferências, pré-prints). Oferece busca full-text (título/resumo) e filtragem por campos (autor, instituição, conceito), com limite alto de requisições. Também provê metadados de periódicos, autores e citações. Não requer chave (só e-mail) e permite acesso aberto aos dados.
* **CORE API** ([https://core.ac.uk/services/api](https://core.ac.uk/services/api)) – Índice de acesso aberto com \~291 milhões de artigos (dois milhões diários). Tem busca rica em filtros (palavra-chave, ano, autoria) e retorna links diretos aos PDFs quando disponíveis. Ideal para recuperar obras de acesso aberto a partir de consultas avançadas.
* **Semantic Scholar API** ([https://api.semanticscholar.org](https://api.semanticscholar.org)) – Motor de busca acadêmico com IA. Permite buscar artigos por título, assunto ou DOI e fornece metadados (citação, autores, referências). Suporta filtros avançados e recomendações de artigos relacionados. Como destaca guia de pesquisa, “o Semantic Scholar API (tem texto completo de acesso aberto como o CORE)”. Ótimo para explorador por citações e grafos de artigos.
* **Unpaywall API** ([https://unpaywall.org/products/api](https://unpaywall.org/products/api)) – API aberta que, dado um DOI, retorna versões de acesso aberto do artigo (links diretos para PDFs gratuitos) sempre que disponíveis. Pode ser integrada para obter o link direto do texto completo ou saber se o artigo está em repositório institucional.
* **Dimensions** ([https://app.dimensions.ai](https://app.dimensions.ai), [https://www.dimensions.ai](https://www.dimensions.ai)) – Plataforma (Digital Science) que cobre cerca de 120 milhões de publicações. Oferece interface web e API paga (há versão gratuita *Lite* para pesquisa limitada) com filtros sofisticados (por ano, autores, área, financiamento, citações, altmetrics). Gera links para artigos via DOI. Bom para análises bibliométricas e achar trabalhos correlacionados.
* **Scopus API / Web of Science API** – APIs comerciais de bases de dados acadêmicas (Elsevier e Clarivate). Cobrem dezenas de milhões de artigos indexados (Scopus \~82M) com busca avançada por autor, citação, assunto. São pagas e exigem credenciais institucionais. São ideais para filtros finos e extração de citações oficiais, mas são restritas a usuários credenciados.
* **arXiv API** ([https://arxiv.org/help/api/user-manual](https://arxiv.org/help/api/user-manual)) – Serviço gratuito de consulta à biblioteca de pré-publicações (física, matemática, ciência da computação, etc.). Suporta **busca avançada** por área, autor, título e operadores booleanos. Retorna metadados e links diretos aos e-prints PDF. Útil para obter rapidamente artigos recentes de acesso aberto em computação e IA.
* **DBLP** ([https://dblp.org/search](https://dblp.org/search)) – Repositório de referência em ciência da computação. Fornece busca por título, autor, ano em vários periódicos e conferências. Disponibiliza API/XML para consulta de publicações de autores ou periódicos específicos. Ideal para encontrar trabalhos de computação e obter referências (embora não hospede os PDFs).
* **Lens.org** ([https://www.lens.org](https://www.lens.org)) – Ferramenta gratuita que integra pesquisa de artigos acadêmicos e patentes. Possui API pública e permite busca avançada por texto completo, com filtros (ano, autor, instituição) e exportação de resultados. Retorna links para publicações via DOI e é útil para pesquisas interdisciplinares.
* **Google Scholar** – Embora não ofereça API oficial, sua interface web de busca permite filtros de pesquisa avançados (por autor, período, periódico, frases exatas). Não recomendado para automação, mas é uma importante referência não-API. Para uso programático, há ferramentas de scraping (como *serpapi*), porém devem respeitar termos de uso.

Cada uma dessas ferramentas oferece documentação (links acima) para consultas programáticas. Em geral, APIs como CrossRef, OpenAlex e Semantic Scholar retornam metadados com DOIs, títulos e resumos, permitindo depois formar URLs diretas (por exemplo `https://doi.org/DOI` ou `https://api.semanticscholar.org/`) para obter conteúdos ou buscar PDFs. Ferramentas como Unpaywall completam o fluxo encontrando o link aberto do PDF quando disponível. Em suma, uma combinação de **CrossRef/OpenAlex/CORE** (grátis) e **Scopus/Dimensions/Semantic Scholar** (pago/avançado) costuma cobrir a maioria das necessidades de busca avançada e obtenção de links diretos a artigos em ciência da computação e IA.
