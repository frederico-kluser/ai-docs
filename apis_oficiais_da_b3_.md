# APIs Oficiais da B3  
- *B3 for Developers – Área do Investidor (oferta pública, posição etc.)* – Portal oficial da B3 (developers.b3.com.br) com APIs como Oferta Pública (dados até D‑1 da participação em ofertas de ações) ([Área do Investidor](https://developers.b3.com.br/apis/api-area-do-investidor#:~:text=B3%20Investidor%20)) e Posição (saldos de investimento até D‑1) ([Área do Investidor](https://developers.b3.com.br/apis/api-area-do-investidor#:~:text=B3%20Investidor%20)). Outras APIs oficiais incluem serviços de custódia (Banco B3) e Tesouro Direto. Dados: participação em IPOs, saldos de contas, eventos corporativos (dividendos, splits, etc.), títulos públicos. Preço: apenas via contrato B2B (pago). Autenticação: credenciais B3 (token/OAuth). Doc oficial: B3 for Developers (ex.: [B3 Investidor](https://developers.b3.com.br/apis/api-area-do-investidor)). Exemplo: curl -H "Authorization: Bearer SEU_TOKEN" "https://developers.b3.com.br/api/investors/offers?date=2025-04-29" (substitua pelo endpoint e token fornecidos pela B3).  

- *Banco B3 – API de Custódia (Fundos)* – API oficial da B3 para fundos de investimento sob custódia do Banco B3 ([APIs](https://developers.b3.com.br/apis#:~:text=Banco%20B3%20,Cust%C3%B3dia)). Dados: boletos de operações, relatórios de posição, validação de cotas de FIIs diárias. Preço: contrato. Autenticação: credenciais B3. Doc: seção “Banco B3 – API de Custódia” no portal B3 (ex.: [documentação](https://developers.b3.com.br)). Exemplo: consulte saldos de FII.  

- *Tesouro Direto (API “Bonds”)* – API oficial da B3 sobre títulos públicos ([APIs](https://developers.b3.com.br/apis#:~:text=v%201.1.0)). Dados: informações sobre títulos públicos (taxas, preços). Preço: contrato. Autenticação: credenciais B3. Doc: “Bonds” no portal B3. Exemplo: curl -H "Authorization: Bearer SEU_TOKEN" "https://developers.b3.com.br/api/tesouro/titulo?isin=BRGP2027ABR5".

> *Obs.:* O *Market Data* (cotações em tempo real, índices, derivativos) não é liberado abertamente pela B3; apenas por distribuidores oficiais (por exemplo, Market Data Cedro) ([Market Data B3: como acessar e quais dados estão disponíveis](https://www.cedrotech.com/blog/market-data-b3-2/#:~:text=Para%20conseguir%20acessar%20os%20dados,como%20o%20Market%20Data%20Cedro)). A B3 disponibiliza historicamente séries históricas em formato de arquivos para download, mas sem API pública direta.

## APIs gratuitas de terceiros  
- *Brapi* – Site: [brapi.dev](https://brapi.dev/) – API brasileira de ações (terceiro). Dados: cotações em atraso (~15 min), históricos (até meses), indicadores, balanços, dividendos de ações/ETFs/BDRs/BDRs/FIIs, índices (ex.: ^BVSP) ([brapi - API de ações da bolsa de valores brasileira](https://brapi.dev/#:~:text=%2A%20,IBOV%29%20pela%20API)) ([brapi - API de ações da bolsa de valores brasileira](https://brapi.dev/pricing#:~:text=)). Preço: Plano gratuito (até 15.000 req/mês, 1 ativo/req, atrasos de ~30 min) ([brapi - API de ações da bolsa de valores brasileira](https://brapi.dev/pricing#:~:text=Pago%20mensalmente)); planos pagos (Startup, Pro) com dados completos. Autenticação: token via cadastro gratuito. Doc: [documentação da brapi](https://brapi.dev/documentation). Exemplo:  
  bash
  curl -X GET "https://brapi.dev/api/quote/ITUB4?range=1y&interval=1mo"
    
  Isso retorna JSON com dados de preço da ação ITUB4 (Itaú Unibanco) ([brapi - API de ações da bolsa de valores brasileira](https://brapi.dev/#:~:text=%24%20curl%20)).

- *Dados de Mercado (DDM)* – Site: [dadosdemercado.com.br](https://www.dadosdemercado.com.br/). Base aberta brasileira integrada (CVM, Anbima, B3 etc.) ([Dados de Mercado - Um banco de dados aberto de investimentos no Brasil](https://www.dadosdemercado.com.br/#:~:text=O%20Dados%20de%20Mercado%20torna,deste%20website%20e%20da%20API)). Dados: empresas listadas (balanços, DRE, fluxo de caixa, indicadores, dividendos, número de ações), cotações de ativos, índices B3 e internacionais, fluxos de investimento, títulos públicos, entre outros ([Documentação da API - Dados de Mercado](https://www.dadosdemercado.com.br/api/docs#:~:text=Empresas%20%20%2023%20,GET%20N%C3%BAmero%20de%20a%C3%A7%C3%B5es)). Preço: plano freemium (é preciso cadastro; há token gratuito limitado e planos pagos para maior volume). Autenticação: header Authorization: Bearer SEU_TOKEN (token obrigatório em todas requisições) ([Documentação da API - Dados de Mercado](https://www.dadosdemercado.com.br/api/docs#:~:text=O%20acesso%20%C3%A9%20feito%20atrav%C3%A9s,%C3%A9%20necess%C3%A1rio%20em%20todas%20as)). Doc: [Documentação DDM](https://api.dadosdemercado.com.br/v1/docs). Exemplo:  
  bash
  curl -H "Authorization: Bearer SEU_TOKEN" "https://api.dadosdemercado.com.br/v1/companies"
    
  Retorna lista de empresas listadas na B3 ([Empresas: Lista de empresas - Documentação da API - Dados de Mercado](https://www.dadosdemercado.com.br/api/docs/empresas/lista-de-empresas#:~:text=Exemplo%20de%20chamada)).

- *HG Finance (HG Brasil)* – Site: [hgbrasil.com](https://hgbrasil.com/). API brasileira de finanças. Dados: cotações e históricos de ações B3, FIIs, BDRs, dólar, Bitcoin, resgates e dividendos de ações ([API preço da ação B3 S.A. - Brasil. Bolsa. Balcão. - B3SA3 cotação - HG Finance](https://hgbrasil.com/apis/cotacao-acao/b3-brasil-bolsa-balcao-b3sa3#:~:text=GET%20https%3A%2F%2Fapi)). Preço: grátis até 400 consultas/dia em “Finance” (bolsa) ([Planos e preços HG APIs - HG Brasil](https://hgbrasil.com/apis/planos#:~:text=Weather%2010%20cidades%20e%201,%2F%20dia)); plano pago (R$12,90/mês) eleva limite. Autenticação: key no parâmetro URL (API Key obtido no console HG). Doc: [Console HG](https://console.hgbrasil.com). Exemplo:  
  bash
  curl -X GET "https://api.hgbrasil.com/finance/stock_price?key=SUA_CHAVE&symbol=B3SA3"
    
  (retorna JSON com dados da ação B3SA3) ([API preço da ação B3 S.A. - Brasil. Bolsa. Balcão. - B3SA3 cotação - HG Finance](https://hgbrasil.com/apis/cotacao-acao/b3-brasil-bolsa-balcao-b3sa3#:~:text=GET%20https%3A%2F%2Fapi)).

- *Alpha Vantage* – Site: [alphavantage.co](https://www.alphavantage.co/) – API global gratuita (terceiro). Dados: dados históricos diários/intraday de ações, forex, cripto, indicadores técnicos; dados fundamentalistas (balanço, DRE, fluxo, estatísticas), calendário de IPOs, etc. Suporta ações brasileiras (ex.: sufixo .SA para B3) ([Original Bovespa's API is dead · Issue #5 · nihey/node-bovespa · GitHub](https://github.com/nihey/node-bovespa/issues/5#:~:text=Alpha%20Vantage%20seems%20to%20be,not%20available%20in%20Alpha%20Vantage)). Preço: gratuito (5 chamadas/minuto, 500/dia) ([Original Bovespa's API is dead · Issue #5 · nihey/node-bovespa · GitHub](https://github.com/nihey/node-bovespa/issues/5#:~:text=Alpha%20Vantage%20seems%20to%20be,not%20available%20in%20Alpha%20Vantage)); planos pagos com mais limite. Autenticação: apikey na URL (token grátis pré-cadastro). Doc: [documentação AlphaVantage](https://www.alphavantage.co/documentation/). Exemplo:  
  bash
  curl "https://www.alphavantage.co/query?function=TIME_SERIES_DAILY&symbol=B3SA3.SAO&apikey=SEU_APIKEY"
    
  (retorna JSON do histórico diário da ação B3SA3).  

- *Twelve Data* – Site: [twelvedata.com](https://twelvedata.com/) – API global (terceiro). Dados: cobertura mundial de ações (incluindo Brasil), ETFs, índices, forex, cripto; séries temporais, indicadores técnicos, estatísticas e dados fundamentalistas (dividendos, balanços, etc). Suporte ao índice Ibovespa (símbolo BOVESPA) ([Brazil Stock Exchange (B3) | Twelve Data Support](https://support.twelvedata.com/en/articles/5070642-brazil-stock-exchange-b3#:~:text=Availability)). Preço: plano gratuito (800 chamadas/dia; 8 símbolos por requisição) ([Pricing - Twelve Data](https://twelvedata.com/pricing#:~:text=800%20%2F%20day)); planos pagos oferecem mais chamadas e menor latência. Autenticação: apikey na URL. Doc: [API docs Twelve Data](https://twelvedata.com/docs). Exemplo:  
  bash
  curl "https://api.twelvedata.com/time_series?symbol=PETR4:Bovespa&interval=1min&apikey=SEU_APIKEY"
    
  (retorna JSON com cotação intraday de PETR4) ([Brazil Stock Exchange (B3) | Twelve Data Support](https://support.twelvedata.com/en/articles/5070642-brazil-stock-exchange-b3#:~:text=Availability)).

## APIs pagas de terceiros  
- *Cedro Technologies – Market Data Web Feeder* – Site: [marketdatacloud.com.br](https://www.marketdatacloud.com.br/) (plataforma Market Data Cedro). Dados: dados de mercado B3 em tempo real (cotações, trades, índices, dólares, commodities), atrasados ou em tempo real, via REST ou streaming ([Como integrar Market Data BM&FBOVESPA (B3) via web services REST em XML/JSON - Cedro Technologies](https://www.cedrotech.com/blog/como-integrar-o-market-data-bmfbovespa-via-servicos-rest-xml-ou-json/#:~:text=Autentica%C3%A7%C3%A3o%20para%20estabelecer%20sess%C3%A3o%3A)). Preço: pago (contrato corporativo; há testes gratuitos de ~7–15 dias). Autenticação: login/senha para sessão (chave de sessão retornada) ([Como integrar Market Data BM&FBOVESPA (B3) via web services REST em XML/JSON - Cedro Technologies](https://www.cedrotech.com/blog/como-integrar-o-market-data-bmfbovespa-via-servicos-rest-xml-ou-json/#:~:text=Autentica%C3%A7%C3%A3o%20para%20estabelecer%20sess%C3%A3o%3A)). Doc: [docs.cedrotech.com](https://docs.cedrotech.com) e blog técnico Cedro. Exemplo:  
  bash
  curl -X POST "http://webfeeder.cedrofinances.com.br/SignIn?login=USUARIO&password=SENHA"
  # recebe token de sessão
  curl -H "Authorization: Bearer TOKEN" "http://webfeeder.cedrofinances.com.br/services/quotes/quote/petr4"
    
  (retorna JSON com cotação de PETR4 em tempo real) ([Como integrar Market Data BM&FBOVESPA (B3) via web services REST em XML/JSON - Cedro Technologies](https://www.cedrotech.com/blog/como-integrar-o-market-data-bmfbovespa-via-servicos-rest-xml-ou-json/#:~:text=Autentica%C3%A7%C3%A3o%20para%20estabelecer%20sess%C3%A3o%3A)).

- *Cedro Technologies – API de Dados Fundamentalistas* – Site: [cedrotech.com](https://www.cedrotech.com/). Dados: indicadores fundamentalistas e demonstrações financeiras consolidadas (base ICVM 358) das empresas brasileiras ([APIs Dados Fundamentalistas - Cedro Technologies](https://www.cedrotech.com/market-apis/apis-dados-fundamentalistas/#:~:text=APIs%20Dados%20Fundamentalistas)). Preço: pago (contrato). Autenticação: fornecida pela Cedro. Doc: [Cedro Fundamental](https://docs.cedrotech.com). Exemplo: acessar balanço, DRE, indicadores financeiros (LPA, ROE etc.) de uma ação via chamadas REST definidas na documentação.

- *Nelogica DataFeed (DLL Real Time)* – Site: [nelogica.com.br](https://www.nelogica.com.br/). Dados: base histórica completa B3 (tick–a–tick) e dados em tempo real via biblioteca DLL. Entrega dados históricos tick de até 90 dias ([Como requisitar dados históricos com a DLL Real Time – Nelogica Sistemas de Software](https://ajuda.nelogica.com.br/hc/pt-br/articles/11973319153563-Como-requisitar-dados-hist%C3%B3ricos-com-a-DLL-Real-Time#:~:text=Neste%20artigo%20vamos%20abordar%20de,de%20qualquer%20ativo%20da%20B3)). Preço: licenciamento corporativo (DataFeed + DLL). Autenticação: acesso via instalação da DLL (token de licenciamento). Doc: [Central de Ajuda Nelogica](https://ajuda.nelogica.com.br). Exemplo: em ambiente Windows/Python, chama-se por ex.: 
  python
  # Exemplo ilustrativo (uso da DLL Nelogica)
  import NelogicaDLL
  hist = NelogicaDLL.GetHistoryTrades("PETR4", inicio, fim)
    
  Isso solicita via DLL dados históricos de negociações (ticks) da ação. Detalhes na documentação da API Nelogica ([Como requisitar dados históricos com a DLL Real Time – Nelogica Sistemas de Software](https://ajuda.nelogica.com.br/hc/pt-br/articles/11973319153563-Como-requisitar-dados-hist%C3%B3ricos-com-a-DLL-Real-Time#:~:text=Neste%20artigo%20vamos%20abordar%20de,de%20qualquer%20ativo%20da%20B3)).

Cada API acima oferece documentação oficial nos respectivos sites (citados) e exige formatos de requisição padrão (JSON/REST). As chamadas de exemplo acima ilustram como obter cotações ou dados básicos usando curl.
